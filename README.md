## **Hangman Game in Python**

A classic command-line Hangman game with word database and scoring system.

### Features
- Random word selection from external file (`words.txt`)
- 6 lives per game
- Input validation and error handling
- Win/loss tracking
- Replay option

### How to Run
```bash
# Clone the repository
git clone https://github.com/maalberto-engineer/hangman_game.git

# Run the game
cd hangman_game
python hangman.py
